# IMS (Integrated Management System)

브라우저 기반 부대 운영 통합 관리 시스템. 별도의 서버 구축 없이 로컬 환경에서 인사, 선박, 근무, 식단 데이터를 통합 관리함.

## 🌟 핵심 가치
*   **서버리스(Serverless)**: 별도의 서버 설치 없이 `index.html` 단독 실행 가능
*   **데이터 보안**: 브라우저 로컬 저장소(IndexedDB)를 사용해 외부 유출 위험 낮음
*   **통합 관리**: 인사, 선박, 근무, 식단, 데이터 관리 기능을 하나의 플랫폼에 통합

---

## 🛠️ 주요 세부 기능

### 1. 인사 정보 관리 (`personnelMattersInformation`)
*   **실시간 복무 현황**: 입대/전역일 기준 총 복무일, 남은 일수, 복무율을 실시간(ms 단위) 산출 및 시각화
*   **자동 진급 계산**: 계급별 진급 예정일 자동 산출 및 다음 계급까지의 잔여 기간 표시
*   **개인 프로필 관리**: 사진(Base64), 소속, 직책, 휴가 일정, 취미/특기 등 상세 데이터 관리
*   **대시보드 기능**: 전체 인원의 핵심 상태를 카드 형태로 한눈에 파악 가능

### 2. 선박 데이터베이스 (`shipDB`, `shipTrace`)
*   **제원 관리**: 선명, 선종, 톤수, 선번호, 연락처 등 상세 제원 데이터베이스화
*   **태그 및 필터링**: 사용자 정의 태그를 통한 선박 분류 및 복합 검색 기능 지원
*   **시각화**: 선박별 다중 사진 등록 및 슬라이더를 통한 식별 용이성 확보

### 3. 식단표 및 근무 관리 (`menu`, `workSchedule`)
*   **지능형 식단 표시**: 시스템 날짜와 연동하여 해당 일자의 조·중·석식 메뉴 자동 출력
*   **근무 스케줄링**: 부대별 특성에 맞춘 근무표 관리 및 확인 기능

### 4. 데이터 관리 시스템 (`databaseManager`)
*   **JSON 백업/복구**: DB 전체 데이터를 JSON 파일로 내보내거나 가져오기 가능하여 데이터 연속성 보장
*   **보안 설정**: 중요 데이터 수정 및 시스템 설정 시 비밀번호 확인 절차 수행

### 5. 엔터테인먼트 모듈 (`game`)
*   **오목(Omok)**: Canvas API 기반의 보드 게임
*   **테트리스(Tetris)**: 고전 게임 로직 구현
*   **생존 게임(Survival)**: 아케이드형 미니게임
*   **지뢰찾기(Minesweeper)**: 퍼즐 게임

---

## 💻 기술적 특징
*   **Frontend**: Vanilla JS (ES6+), HTML5 Canvas, CSS3 (Grid/Flexbox)
*   **Database**: IndexedDB를 활용한 로컬 데이터 영속성 확보
*   **UI/UX**: 모달 시스템, 반응형 카드 레이아웃 적용

## 🚀 실행 방법
1.  저장소 다운로드 또는 클론
2.  브라우저에서 `index.html` 실행
3.  **데이터베이스 관리** 메뉴에서 기초 데이터 입력 또는 백업 파일 임포트
